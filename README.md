# GTTC Hubli — Lead CRM

Lead management dashboard for GTTC Hubli (Classes 8–10 English program CRM).

## Run locally

```bash
npm install
npm run dev
```

Open http://localhost:5173

## Build

```bash
npm run build
```

## Project structure

- `src/components/` — Dashboard, layout, charts
- `src/data/sampleLeads.ts` — Sample lead data
- `src/types/lead.ts` — Lead types
- `src/utils/leadStats.ts` — Dashboard calculations
